# Plano de Ensino

## Curso de Desenvolvimento com InterSystems IRIS

## 1\. Competências

Ao final do curso, o participante será capaz de:

- Desenvolver aplicações utilizando programação orientada a objetos com ObjectScript moderno;  
- Projetar e implementar integrações utilizando os componentes de Interoperability (BS, BP, BO e Adapters);  
- Aplicar princípios de engenharia de software, como SOLID e padrões de design;  
- Compreender pipelines básicos de integração e entrega contínua (CI/CD);  
- Desenvolver e executar testes automatizados (unitários e de integração);  
- Diagnosticar e resolver problemas com técnicas avançadas de debugging;  
- Trabalhar de forma colaborativa utilizando Git e GitHub.

---

## 2\. Habilidades

Os participantes irão desenvolver as seguintes habilidades:

- Uso avançado do Visual Studio Code para desenvolvimento com IRIS;  
- Manipulação de código entre múltiplos namespaces;  
- Controle de versão com Git (branching, commits, merge, pull requests);  
- Colaboração em equipe com GitHub (PRs, code review);  
- Desenvolvimento orientado a objetos com boas práticas;  
- Desenvolvimento de integrações síncronas e assíncronas;  
- Escrita de testes automatizados;  
- Análise crítica de código e revisão de qualidade (code review);  
- Troubleshooting em produções IRIS.

---

## 3\. Conteúdo Programático

### 3.1 Fundamentos e Setup do Ambiente

- Introdução ao curso e ambiente de desenvolvimento;  
- Instalação e configuração do Visual Studio Code;  
- Configuração de extensões para InterSystems IRIS;  
- Conceitos de workspace e configuração de ambiente.

---

### 3.2 Desenvolvimento com IRIS

- Programação orientada a objetos com ObjectScript;  
- Boas práticas de desenvolvimento e padronização;  
- Manipulação de código entre namespaces;  
- Deploy manual e gerenciamento de código no servidor.

---

### 3.3 Controle de Versão e Colaboração

- Conceitos de versionamento;  
- Fluxo de trabalho com Git (local e remoto);  
- Branches, commits e staging;  
- Integração com GitHub;  
- Pull Requests e Code Review;  
- Boas práticas de colaboração em equipe.

---

### 3.4 Qualidade de Código

- Introdução a Style Guides;  
- Princípios SOLID;  
- Análise e revisão de código;  
- Uso de ferramentas de formatação.

---

### 3.5 Interoperability (Integrações)

- Introdução ao InterSystems Interoperability;  
- Componentes: Business Service (BS), Business Process (BP), Business Operation (BO);  
- Estruturas de dados e mensagens (Request/Response);  
- Modelagem de integrações;  
- Definição de responsabilidades em fluxos.

---

### 3.6 Projeto Prático

Desenvolvimento de um projeto completo envolvendo:

- Fluxo ETL (extração, transformação e carga);  
- API síncrona de consulta;  
- API assíncrona com processamento em fila;  
- Versionamento e colaboração;  
- Debugging e correção de erros;  
- Evolução incremental com branches.

---

### 3.7 Testes e Qualidade

- Testes unitários;  
- Testes de integração;  
- Estratégias de testes.

---

### 3.8 DevOps e Entrega

- Conceitos de CI/CD;  
- Integração com pipelines;  
- Processo de deploy baseado em Git;  
- Sincronização de ambientes.

---

## 4\. Metodologia

O curso será conduzido utilizando uma abordagem prática e colaborativa:

- **Aulas expositivas:** apresentação dos conceitos fundamentais;  
- **Exercícios práticos:** aplicação imediata dos conceitos;  
- **Pair programming:** desenvolvimento em duplas;  
- **Projeto colaborativo:** desenvolvimento de solução completa;  
- **Code review:** revisão estruturada de código entre participantes;  
- **Mentoria contínua:** acompanhamento e feedback ao longo do curso.

---

## 5\. Recursos Necessários

- Ambiente InterSystems IRIS (local ou remoto);  
- Visual Studio Code configurado;  
- Conta no GitHub;  
- Repositório base do curso;  
- Materiais de apoio (documentação, style guide, guias práticos).
