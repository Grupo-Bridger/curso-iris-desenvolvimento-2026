# Guia do autor da CL para passar pela revisão de código

As páginas desta seção contêm práticas recomendadas para desenvolvedores que
estão passando por revisão de código. Estas diretrizes devem ajudar você a
passar pelas revisões mais rapidamente e com resultados de maior qualidade. Você
não precisa ler tudo, mas elas foram pensadas para se aplicar a todo
desenvolvedor do Google, e muitas pessoas acharam útil ler o conjunto inteiro.

-   [CLs pequenas](small-cls.md)
-   [Como lidar com comentários do revisor](handling-comments.md)

orientação detalhada para revisores de código.
